How to use Guild Tracker

  1. Log in or register Use Login in the header, or go to /login. On that page, use Register if you need an account.
  2. Find a guild Click Search Guild below, or open /guild/search. Search by guild name or prefix.
  3. Track and start an event From the guild card, click Select Players, choose members, set metric and scope, then Start Event.
  4. Join or review events Use Enter Event Code to join by code, Event History for past runs, and Open Leaderboard while an event is active.
